ARCS(S) Motivational Design Model

supports the creation of instruction that:

  • gains/sustains a learner’s interest → attention
  • includes personal-motive value for the learner → relevance
  • provides personal control over success → confidence
  • provides informative feedback → satisfaction
  • promotes continued interest (volition) → self-motivation

Instructional Design

  • There are many, many models & methods.
  • They are meant to be repeatable & process-driven.
  • Many/most tend to address (Bloom's) cognitive domain objectives.
  • Keller’s ARCS(S) model addresses motivational aspects of learning.
  • These are addressed within Bloom's affective domain objectives.

The Process

  1. Identify any motivational issues associated with the instruction.
  2. Brainstorm motivational strategies using the ARCS(S) model.
  3. Implement your revisions.
  4. Evaluate the effects of the changes.
